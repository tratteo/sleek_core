
# Sleek Core

## Quick Setup

1. Install the module to your Nuxt application with one command:

```bash
npm install github:tratteo/sleek_core
```
2. Add the module to your `nuxt.config.ts`:
```ts
modules: [
  "@sleek/core", 
  /*... other modules */
],
```
